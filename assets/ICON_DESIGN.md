# Tag Navigator icon

The mark combines a library tag with a compass needle: tags are the material
being explored and the compass communicates navigation. The Zotero-inspired red
keeps it visually related to the host application without reproducing Zotero's
lettermark.

## Source

- Master with transparency: `tag-navigator-icon-master.png`
- Runtime exports: `addon/content/icons/tag-navigator-{16,32,48,96,128}.png`

## Generation prompt

> Create one original desktop application icon for Tag Navigator: a bold
> library-tag silhouette fused with a minimal compass needle. Use a clean,
> vector-like flat style, precise geometry, a strong silhouette, balanced
> negative space and generous padding. Use Zotero-inspired deep red, warm white
> and dark charcoal only if essential. No text, letters, watermark, 3D, cast
> shadow or reflection. Generate it on a uniform green chroma-key background for
> local transparency removal.

The source was generated with the built-in image-generation tool. Its chroma
background was removed locally and the runtime sizes were exported with Lanczos
resampling.
